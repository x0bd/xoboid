Glyph is a modern JavaScript frontend framework that is heavily inspired by React, Svelte and Vue. I am building the framework to better understand how these engineering marvels work under the hood.

##### Tech Stack

-   JavaScript
-   Rollup
-   NPM
-   Vitest
